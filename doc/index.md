# Cognitive Robot Abstract Machine (CRAM)

Monorepo for the CRAM cognitive architecture.

This documentation serves as a central hub for all sub-packages within the CRAM ecosystem.

## Sub-packages

- [pycram](pycram): Cognitive Robot Abstract Machine in Python.
- [krrood](krrood): Knowledge Representation and Reasoning for Object-Oriented Domains.
- [semantic_digital_twin](semantic_digital_twin): Semantic Digital Twin for Robotics.
- [probabilistic_model](probabilistic_model): Probabilistic Modeling for Robotics.
- [random_events](random_events): Random Events and Processes.
- [giskardpy](https://github.com/SemRoCo/giskardpy): Motion planning and control for robots.
