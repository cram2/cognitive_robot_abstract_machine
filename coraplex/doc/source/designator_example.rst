===========
Designators
===========

The following notebooks demonstrate the concrete designator types in CoraPlex and how to use them in a plan. For the
conceptual background on what designators are and how they are grounded, see :doc:`designators`.

The examples cover:

* **Action designators** -- high-level tasks such as navigating, picking up and placing.
* **Object designators** -- referencing bodies in the belief state, including querying them with the Entity Query
  Language.
* **Location designators** -- describing feasible poses for visibility, reachability and occupancy via costmaps.
* **Motion designators** -- the low-level movements that actions are built from.
